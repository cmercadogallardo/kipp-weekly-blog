# 🔍 KIPP Weekly Blog

**Conocimiento Semanal, Análisis Profundo**

Blog estático generado automáticamente con las investigaciones y briefings semanales de KIPP (Knowledge & Information Processing Partner).

## 📊 Estructura

```
kipp-weekly-blog/
├── index.html              # Landing page principal
├── indice-mensual.html     # Índice mensual de briefings
├── entrada-blog.html       # Template para entradas individuales
├── styles.css              # Estilos personalizados KIPP
├── main.js                 # Funcionalidades JavaScript
└── semanas/                # Contenido semanal
    ├── semana-1.html
    ├── semana-1-dia-1.html
    ├── semana-1-dia-2.html
    └── ...
```

## 🚀 GitHub Pages

Este sitio está hospedado en GitHub Pages:

**URL:** https://cmercadogallardo.github.io/kipp-weekly-blog/

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 moderno (variables, flexbox, grid)
- JavaScript vanilla (sin frameworks)
- GitHub Pages para hosting

## 📝 Actualización Automática

El contenido se genera automáticamente cada semana mediante un proceso automatizado que:
1. Convierte briefings de Markdown a HTML
2. Actualiza índices semanales y mensuales
3. Hace commit y push al repositorio
