04 - Prompts de Desarrollo UI

Fecha: 2026-05-19
Investigador: KIPP - Líder Investigador
Objetivo: Generar prompts detallados para cada sección del sitio

Guía de Uso

Estos prompts están diseñados para: - Herramientas de diseño: Figma, Adobe XD, Sketch - IA generativa: Midjourney, DALL-E 3, Stable Diffusion (para mockups) - Desarrolladores: Especificaciones técnicas claras - Frameworks: Tailwind CSS, Bootstrap, o desarrollo custom

Estilos Globales (Design System)

Paleta de Colores

COLOR PALETTE - CUÁNTICA PYD

Primarios:
- Azul Profundo: #1E3A5F (confianza, profesionalismo)
- Azul Acero: #4A6FA5 (secundario, acciones)

Neutros:
- Blanco Puro: #FFFFFF (fondos)
- Gris Claro: #F5F7FA (fondos secundarios)
- Gris Medio: #9CA3AF (texto secundario)
- Gris Oscuro: #374151 (texto principal)
- Negro Suave: #1F2937 (títulos, contraste)

Acentos:
- Dorado Arena: #C9A961 (lujo, premium)
- Terracota: #B87355 (calidez, materiales naturales)

Estados:
- Éxito: #10B981
- Advertencia: #F59E0B
- Error: #EF4444
- Info: #3B82F6

Tipografía

TYPOGRAPHY SYSTEM

Títulos (Display):
- Familia: "Playfair Display" o "Montserrat"
- Pesos: 600 (SemiBold), 700 (Bold)
- Uso: H1, H2, H3

Cuerpo (Body):
- Familia: "Inter" o "Open Sans"
- Pesos: 400 (Regular), 500 (Medium)
- Uso: Párrafos, descripciones

Tamaños (Desktop):
- H1: 48-56px
- H2: 36-42px
- H3: 24-28px
- H4: 20-22px
- Body: 16-18px
- Small: 14px

Tamaños (Móvil):
- H1: 32-36px
- H2: 26-30px
- H3: 20-22px
- Body: 16px
- Small: 14px

Espaciado de línea:
- Títulos: 1.2
- Cuerpo: 1.6

Espaciado y Grid

SPACING SYSTEM (Base: 8px)

xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
3xl: 64px
4xl: 96px

GRID SYSTEM:
- Desktop: 12 columnas, 1140px máx, 24px gutter
- Tablet: 8 columnas, 720px máx, 24px gutter
- Móvil: 4 columnas, 100%, 16px gutter

Componentes UI

BUTTONS:
- Primary: Azul Profundo (#1E3A5F), texto blanco, 48px height, 24px padding-x
- Secondary: Borde Azul Profundo, texto Azul Profundo, fondo transparente
- Hover: Elevar 2px, sombra suave, lighten 10%
- Border radius: 4px (ligeramente redondeado)

CARDS:
- Background: Blanco
- Shadow: 0 4px 6px rgba(0,0,0,0.1)
- Border radius: 8px
- Padding: 24px
- Hover: Shadow 0 8px 15px rgba(0,0,0,0.15)

INPUTS:
- Border: 1px sólido #D1D5DB
- Border focus: 2px sólido #4A6FA5
- Height: 48px
- Border radius: 4px
- Padding: 12px 16px

SECCIÓN 1: HEADER (Sticky Navigation)

Prompt para Diseño

DESIGN PROMPT - HEADER STICKY

Crear navegación sticky para sitio arquitectónico premium.

COMPOSICIÓN:
- Logo a la izquierda (200x50px espacio)
- Menú horizontal centro: "Inicio | Productos | Proyectos | Contacto"
- Botón CTA derecha: "COTIZAR" + ícono teléfono

ESTILO:
- Fondo: Transparente en top → Blanco sólido al scroll
- Altura: 80px
- Tipografía menú: Inter Medium 16px, #374151
- Hover menú: Azul Profundo #1E3A5F, underline 2px
- Botón CTA: Azul Profundo fondo, blanco texto, 48x48px

ESTADOS:
- Default: Transparente
- Scrolled: Blanco #FFFFFF con sombra suave
- Mobile: Hamburguesa derecha, menú drawer

REFERENCIAS VISUALES:
- Minimalista, limpio, profesional
- Similar a sitios de arquitectura de lujo
- Espaciado generoso, aireado

ENTREGABLES:
- Desktop (1440px)
- Tablet (768px)
- Móvil (375px)
- Estados: default, scrolled, hover

- Minimalismo con impacto visual

ENTREGABLES:
