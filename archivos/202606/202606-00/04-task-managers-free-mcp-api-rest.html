🔍 Task Managers FREE Forever con MCP / API REST para Individuos

Fecha: 2026-06-04
Investigación: Task managers gratuitos con capacidades MCP o API REST para uso individual
Excluido: Trello (por solicitud)

📊 Resumen Ejecutivo

Se identificaron 3 opciones principales que cumplen con los criterios: 1. Super Productivity - Mejor opción overall (local-first + API REST + MCP) 2. Vikunja - Mejor opción self-hosted (API REST completa + MCP servers) 3. Taskwarrior - Opción CLI para power users

🏆 Top Recomendaciones

0. Plane 🆕 MEJOR PARA EQUIPOS / AGILE

Sitio: https://plane.so
GitHub: https://github.com/makeplane/plane
Licencia: AGPL-3.0 (Open Source)

✅ Características Clave

Característica Detalle
Precio Gratis (self-hosted) o cloud freemium
Modelo Server-based (cloud o self-hosted)
API REST ✅ Sí, completa y documentada
MCP ✅ Sí, MCP server oficial nativo
Código Abierto ✅ AGPL-3.0
Telemetría ⚠️ Cloud sí, self-hosted no
Plataformas Web, móvil (PWA)

🔌 Integraciones y API

  • REST API: Completa, todas las funciones expuestas
  • MCP Server Oficial: https://mcp.plane.so
    • 100+ herramientas en 20 módulos
    • 4 transport modes: HTTP OAuth, HTTP PAT, Stdio, SSE
    • Soporte nativo: Claude Desktop, Claude Code, Cursor, VS Code, Windsurf, Zed
  • Vistas: Listas, Kanban, Sprint cycles, Modules, Docs, Triage
  • Features: Dashboards, estimates, webhooks, OAuth apps
  • AI-native: Integración con AI para generación de contenido

🎯 Ideal Para

  • Equipos que vienen de Jira/Linear/ClickUp
  • Sprints ágiles y gestión de ciclos
  • Developers que quieren MCP nativo en su editor
  • Proyectos con necesidad de documentación integrada

⚠️ Consideraciones

  • Más complejo que necesario para individuos simples
  • Cloud tiene límites en free tier
  • Self-hosting requiere más recursos que Vikunja
  • Enfoque principal es equipos, no individuos solos

1. Super Productivity ⭐ MEJOR PARA INDIVIDUOS

Sitio: https://super-productivity.com
GitHub: https://github.com/johannesjo/super-productivity
Licencia: MIT (Free Forever)

✅ Características Clave

Característica Detalle
Precio 100% gratis, sin premium tier
Modelo Local-first (tus datos en tu dispositivo)
API REST ✅ Sí, documentada
MCP ✅ Sí, plugins MCP disponibles
Código Abierto ✅ MIT License
Telemetría ❌ Ninguna
Plataformas Windows, macOS, Linux, Android, iOS

🔌 Integraciones y API

  • REST API: Expuesta para automatizaciones personalizadas
  • Plugin System: Sistema de plugins para extender funcionalidad
  • MCP Plugins: Soporte nativo para Model Context Protocol
  • Issue Trackers: GitHub, GitLab, Jira, Azure DevOps, ClickUp, Linear, Redmine, Gitea, OpenProject
  • Sync: WebDAV, Dropbox, Nextcloud Deck, CalDAV, Google Calendar
  • Super Sync: Sync opcional con E2EE (AES-256-GCM, Argon2id)

🎯 Ideal Para

  • Developers que quieren time tracking integrado
  • Individuos que priorizan privacidad
  • Quienes necesitan integraciones con GitHub/Jira
  • Usuarios que quieren control total de sus datos

⚠️ Consideraciones

  • Colaboración en equipo limitada (diseñado para individuos)
  • Mantenedor principal único (Johannes Millan)
  • Sin auditoría de seguridad third-party

2. Vikunja 🥈 MEJOR SELF-HOSTED

Sitio: https://vikunja.io
GitHub: https://github.com/go-vikunja/vikunja
Licencia: AGPL / Apache 2.0

✅ Características Clave

Característica Detalle
Precio Gratis (self-hosted) o cloud pago opcional
Modelo Server-based (tú controlas el server)
API REST ✅ Sí, completa y documentada
MCP ✅ Sí, múltiples MCP servers disponibles
Código Abierto ✅ AGPL / Apache 2.0
Telemetría ❌ Ninguna (self-hosted)
Plataformas Web, móvil (vía navegador)

🔌 Integraciones y API

  • REST API: Completa, todas las funciones expuestas
  • MCP Servers Disponibles:
    • vikunja-mcp - PyPI official
    • aimbitgmbh/vikunja-mcp - GitHub
    • 0xK3vin/vikunja-mcp - 32 tools, 90%+ coverage
  • Vistas: Listas, Kanban, Gantt, Tablas, Mente map
  • Import: Todoist, Trello, JSON
  • Notificaciones: Email, webhooks, matriz federation

🎯 Ideal Para

  • Usuarios técnicos que pueden self-hostear
  • Quienes necesitan vistas múltiples (Kanban, Gantt)
  • Proyectos personales complejos
  • Posible escalado a equipos pequeños

⚠️ Consideraciones

  • Requiere hosting propio (Docker recomendado)
  • Curva de aprendizaje más alta
  • Cloud hosting tiene costo si no self-hosteas

🐳 Docker Quickstart

docker run -d \
  -e VIKUNJA_DATABASE_TYPE=sqlite \
  -e VIKUNJA_DATABASE_PATH=/var/lib/vikunja/vikunja.db \
  -v vikunja-data:/var/lib/vikunja \
  -p 3456:3456 \
  vikunja/vikunja

💡 Insight Principal

Super Productivity es la mejor opción para individuos porque:

  1. Free forever real - Sin premium tier, sin límites artificiales
  2. Local-first - Tus datos nunca salen de tu dispositivo a menos que tú quieras
  3. API + MCP - Extensible para automatizaciones y AI workflows
  4. Time tracking integrado - No necesitas Toggl u otra herramienta
  5. Integraciones dev - GitHub, GitLab, Jira nativos
  6. Privacidad - Sin telemetría, sin mining de datos

Vikunja es mejor si: - Quieres self-hostear todo tu stack - Necesitas vistas Kanban/Gantt - Planeas escalar a equipo pequeño

Taskwarrior es mejor si: - Vives en la terminal - Quieres scripting máximo - Minimalismo extremo
