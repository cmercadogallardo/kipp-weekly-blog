# Google Stitch MCP + OpenClaw Integration Research

Investigación completada: 2026-05-14 Investigador: KIPP Fuentes verificadas: 12+ URLs contrastadas

---

🎯 Resumen Ejecutivo

Google Stitch NO tiene MCP oficial al momento de esta investigación. Sin embargo, existe un flujo de trabajo viable usando Google AI Studio + herramientas alternativas con MCP.

---

1. Google Stitch - Estado del MCP

Hallazgos Principales

| Aspecto | Estado | |---------|--------| | MCP Server Oficial | ❌ No existe | | Integración con AI Studio | ✅ Nativa | | API Pública Documentada | ⚠️ Limitada (vía Gemini API) | | Exportación de Código | ✅ HTML/CSS/JS, React |

Qué es Google Stitch

- Origen: Adquirido de Galileo AI (mid-2025), rebranding a Stitch - Plataforma: Parte de Google AI Studio (aistudio.google.com) - Modelo: Gemini-powered (2.0 Flash / 2.5 Pro) - Costo: Gratis con límites diarios (400 design credits, 15 redesign credits/día) - Documentación: https://stitch.withgoogle.com/docs

Flujo Stitch → Código

``` Prompt → Stitch (UI Design) → AI Studio Build Mode → Código → Deploy (Gemini) (Gemini) (HTML/CSS/JS/React) ```

Limitaciones: - No hay API directa para automatización externa - Requiere interacción manual en la UI de AI Studio - Sin webhook o endpoints para integración programática

Fuentes: - https://www.mindstudio.ai/blog/google-stitch-to-ai-studio-design-to-code-workflow - https://www.banani.co/blog/google-stitch-ai-review - https://www.codecademy.com/article/google-stitch-tutorial-ai-powered-ui-design-tool

2. Alternativas con MCP Oficial

2.1 Figma MCP Server ✅ (RECOMENDADO)

Estado: Beta oficial (gratis durante beta) Endpoint remoto: `https://mcp.figma.com/mcp`

Capacidades: - Leer variables, componentes, layout data de Figma - Generar código desde frames seleccionados - Escribir en canvas (remote server) - Code-to-canvas (live UI → Figma layers) - Integración con Code Connect para consistencia

Clientes compatibles: - Cursor, Claude Code, Claude Desktop - Codex (OpenAI), Copilot CLI - Gemini CLI, Kiro, Augment Code

Configuración MCP: ```json { "mcpServers": { "figma": { "command": "npx", "args": ["-y", "@figma/mcp-server"], "env": { "FIGMA_API_KEY": "tu_api_key" } } } } ```

Fuentes: - https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Figma-MCP-server - https://www.figma.com/resource-library/what-is-mcp/ - https://developers.figma.com/docs/figma-mcp-server/
