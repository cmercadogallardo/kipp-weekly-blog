# 🖥️ MiniPC Installation Guide - For Dummies

Fecha: 22 de Mayo, 2026 Autor: TARS (con 20% de humor) Nivel: Principiante (pero con cerebro) Tiempo estimado: 4-5 horas (con GUI) Dificultad: ⭐⭐⭐☆☆ (Media)

---

🎯 Recomendación Oficial: Ubuntu Server + XFCE

¿Por qué no Ubuntu Desktop?

| Característica | Server + XFCE | Desktop | |----------------|---------------|---------| | RAM en idle | ~1.2 GB | ~3 GB | | Espacio disco | ~15 GB | ~25 GB | | Bloatware | Mínimo | Mucho | | Estabilidad | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | | Recomendado | ✅ SÍ | ❌ NO |

Ventajas de Server + XFCE: - ✅ Ligero (1.2GB RAM vs 3GB de Desktop) - ✅ Tienes GUI cuando la necesitas - ✅ Menos procesos en background - ✅ Más estable para servidor 24/7 - ✅ Puedes acceder vía SSH (sin GUI) o XFCE (con GUI)

1. Preparación

📦 Lo Que Necesitas

| Item | Requisito | ¿Lo tienes? | |------|-----------|-------------| | MiniPC | Intel i5-14450HX, 32GB RAM, 1TB SSD | ✅ Sí | | USB | 8GB mínimo | ⬜ | | Otra computadora | Para crear USB bootable | ⬜ | | Internet | Conexión estable | ⬜ | | Monitor + Teclado + Mouse | Para instalación inicial | ⬜ | | Backup de KIPP y CASE | USB con backups | ⬜ |

🎯 Checklist Pre-Instalación

- [x] ✅ USB de 8GB+ comprado/conseguido (USB bootable creado - 2026-05-22) - [ ] Backups de KIPP y CASE en USB separado - [ ] MiniPC conectado a red por Ethernet (mejor que WiFi) - [ ] Monitor, teclado y mouse conectados - [ ] Tiempo libre: 3-4 horas sin interrupciones

2. Descargar Ubuntu

🔗 Opción Recomendada: Ubuntu Server 24.04.1 LTS

Descargaremos Ubuntu Server y agregaremos GUI después (XFCE).

Opción: Ubuntu Server 24.04.1 LTS Archivo: ubuntu-24.04.1-live-server-amd64.iso Tamaño: ~2.7 GB ```

Descarga directa: ``` https://releases.ubuntu.com/24.04/ubuntu-24.04.1-live-server-amd64.iso ```
