Se identificaron 3 opciones principales que cumplen con los criterios: 1. Super Productivity - Mejor opción overall (local-first + API REST + MCP) 2. Vikunja - Mejor opción self-hosted (API REST completa + MCP servers) 3. Taskwarrior - Opción CLI para power users

🔌 Integraciones y API

- REST API: Completa, todas las funciones expuestas - MCP Server Oficial: https://mcp.plane.so - 100+ herramientas en 20 módulos - 4 transport modes: HTTP OAuth, HTTP PAT, Stdio, SSE - Soporte nativo: Claude Desktop, Claude Code, Cursor, VS Code, Windsurf, Zed - Vistas: Listas, Kanban, Sprint cycles, Modules, Docs, Triage - Features: Dashboards, estimates, webhooks, OAuth apps - AI-native: Integración con AI para generación de contenido

🎯 Ideal Para

- Equipos que vienen de Jira/Linear/ClickUp - Sprints ágiles y gestión de ciclos - Developers que quieren MCP nativo en su editor - Proyectos con necesidad de documentación integrada

- Más complejo que necesario para individuos simples - Cloud tiene límites en free tier - Self-hosting requiere más recursos que Vikunja - Enfoque principal es equipos, no individuos solos
