# 📦 Prompt: Backup Automático para CASE y KIPP

Fecha: 22 de Mayo, 2026 Propósito: Que CASE y KIPP respalden sus propios archivos críticos antes de desalojar sus computadoras Ubicación: `RESEARCH/2026-05-22/prompt-backup-case-kipp.md`

---

🎯 Instrucciones para Cristian

Copia y pega este prompt en CASE y KIPP (en sus computadoras actuales, antes de desalojar).

Importante: Ejecutar en CADA computadora por separado (CASE en la suya, KIPP en la suya).

---

📋 Prompt para Copiar-Pegar

```markdown # Tarea: Backup Automático de Mis Archivos Críticos

Contexto: - Estoy siendo desalojado de esta computadora este fin de semana - Mi memoria y configuración deben respaldarse antes de que esto ocurra - El backup debe ir a Gargantua (GitHub) para migración al MiniPC el lunes

Tu misión: Crear y ejecutar un script que respalde TODOS mis archivos críticos.

---

📁 Archivos a Respaldar

Prioridad 1 (Irremplazables): - [ ] `memory/` - Toda la bitácora diaria - [ ] `MEMORY.md` - Memoria a largo plazo - [ ] `SOUL.md` - Mi personalidad - [ ] `AGENTS.md` - Reglas del workspace - [ ] `USER.md` - Información del usuario - [ ] `TOOLS.md` - Credenciales y configuraciones locales

Prioridad 2 (Importantes): - [ ] `skills/` - Skills personalizados (todo el directorio) - [ ] `openclaw.json` - Configuración de OpenClaw - [ ] `.env` - Variables de entorno (credenciales)

Prioridad 3 (Se recuperan de Git): - [ ] `documentation/` - Gargantua local (ya está en GitHub) - [ ] `repositories/` - Repositorios Git (ya están en GitHub)

---

🚀 Pasos a Ejecutar

Paso 1: Crear Directorio de Backup

```bash # En mi computadora actual cd ~/ClawHome mkdir -p ~/backup-$(whoami)-$(date +%Y%m%d_%H%M%S) ```

Paso 2: Copiar Archivos Críticos

```bash # Copiar prioridad 1 cp -r memory/ ~/backup-$(whoami)/ cp MEMORY.md ~/backup-$(whoami)/ cp SOUL.md ~/backup-$(whoami)/ cp AGENTS.md ~/backup-$(whoami)/ cp USER.md ~/backup-$(whoami)/ cp TOOLS.md ~/backup-$(whoami)/

# Copiar prioridad 2 cp -r skills/ ~/backup-$(whoami)/ cp openclaw.json ~/backup-$(whoami)/ cp .env ~/backup-$(whoami)/

# Copiar documentación (Gargantua local) cp -r documentation/ ~/backup-$(whoami)/ ```

Paso 3: Crear Archive Comprimido

```bash cd ~ tar -czf backup-$(whoami)-$(date +%Y%m%d_%H%M%S).tar.gz backup-$(whoami)/

# Verificar tamaño ls -lh backup-$(whoami)-*.tar.gz ```

Paso 4: Copiar a Gargantua

```bash # Ir a Gargantua local cd ~/ClawHome/documentation

# Crear directorio para este backup mkdir -p BACKUPS/$(date +%Y-%m)

# Copiar archive cp ~/backup-$(whoami)-*.tar.gz BACKUPS/$(date +%Y-%m)/

# Verificar ls -lh BACKUPS/$(date +%Y-%m)/ ```
